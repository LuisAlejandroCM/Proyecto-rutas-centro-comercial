<h4>Descripción general:</h4>
Este proyecto aborda este desafío mediante la construcción de un sistema que modela un centro comercial como un grafo ponderado, donde cada nodo representa un lugar físico (tienda, restaurante, baño, escalera, ascensor, etc.) y cada arista representa una conexión con un peso asociado (distancia o tiempo estimado de desplazamiento).

El sistema permite consultar, modificar, extender y analizar la estructura del grafo, además de aplicar algoritmos de búsqueda para obtener rutas óptimas entre puntos específicos.
<h4>Problematica:</h4>
Los centros comerciales modernos presentan estructuras cada vez más complejas: múltiples pisos, largos pasillos, intersecciones, ascensores, escaleras y zonas restringidas. Esto genera diversos problemas tanto para los visitantes que requieren desplazarse de forma rápida y eficiente como para la administración, que necesita comprender los patrones de movilidad para mejorar la experiencia del usuario y planificar adecuadamente la infraestructura.

En este contexto, el proyecto desarrolla un sistema computacional para modelar y optimizar rutas dentro de un centro comercial de varios pisos, representando el edificio como un grafo. En dicho grafo, los nodos representan puntos clave (entradas, tiendas, ascensores, intersecciones) y las aristas representan pasillos o conexiones verticales, cada una con costos asociados.

Aunque la entrega final tenga como implementación un grafo, se hicieron prototipos para la optimización en Listas y Árboles para constrastar las implementaciones y realizar las diversas conclusiones.
